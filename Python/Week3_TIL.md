**기능**

- 배열 데이터를 디스크에 쓰거나 읽을 수 있는 도구의 메모리에 적재된 파일을 다루는 도구

- 저수준 언어로 작성된 외부 라이브러리에 데이터를 전달하기에 용이 



**넘파이의 장점**

- 내부적으로 데이터를 다른 내장 파이썬 객체와는 구분된 연속된 메모리를 블록에 저장
- 각종 알고리즘이 모두 C로 작성 타입 검사나 다른 오버헤드 없이 메모리 직접 조작이 가능
  - 일반 파이썬 코드 실행에서 발생하는 오버헤드를 줄여서 빠르게 작동이 가능
- **내장 파이썬의 연속된 자료형보다 훨씬 더 작은 메모리를 사용**



## 1. 다차원 배열 객체 ndarray

### 개념정리

**ndarray : N 차원의 배열 객체**

- 파이썬에서 사용하는 대규모 데이터셋을 담을 수 있는 유연한 자료구조 
  - 같은 종류의 데이터를 담는 포괄적인 다차원 배열 

- **생성 방법** : array 함수 사용

- `numpy.array` 를 활용 : 새로운 배열 여러 함수 가지고 있다. 
  - empty 메서드를 사용하여 초기화 되지 않은 배열을 생성이 가능
  - 데이터를 새로채우기 위해 배열을 생성하는 경우에만 이 함수를 사용해야 함.
- **배열 생성 함수**
  - asarray : 입력 데이터를 ndarray로 변환, 입력 데이터가 이미 ndarray인 경우 복사 X
  - Orange : 내장 range 함수와 유사하지만 리스트 대신 ndarray를 반환



**ndarray의 자료형**

- `dtype` : ndarray 가 메모리에 있는 특정 데이터를 해석하는데 필요한 정보를 담는 객체
  - 주로 사용하는 일반적인 타입 : 부동소수점, 복소수, 정수, 불리언, 문자열, 일반 파이썬 객체 존재
  - `astype` 메서드를 통해 다른 형으로 명시적 **변환**이 가능하다.
- 넘파이에서 문자열 데이터는 고정 크기를 가지며 경고를 따로 출력하지 않는다. 



**넘파이 배열의 산술 연산**

- 배열은 for 문을 작성하지 않고 데이터를 처리할 수 있다. **벡터화** 
  - 크기가 동일한 배열 간의 산술 연산이 가능 
  - 스칼라 인수가 포함된 산술 연산 시, 모든 원소에 스칼라 인수가 적용된다. 



**색인과 슬라이싱 기초** : 다차원 배열 시 

- 개별 원소는 재귀적으로 접근해야함. (색인 리스트 사용 가능)
- 넘파이 배열의 다차원 색인 구문은 리스트의 리스트 같은 일반적인 파이썬 객체에서는 작동을 하지 않는다.



**불리언 값으로 선택하기**

- 불리언 배열은 반드시 색인하려는 축의 길이와 길이가 동일해야 한다. 
  - 불리언 배열을 슬라이스나 정수를 선택하는데 짜 맞출 수 있다. 
  - 파이썬 예약어 `and / or` 직접 사용 X **(&, |)** 사용 해야 한다. 



**팬시 색인** : 정수 배열을 사용한 색인을 사용하기 위해 넘파이에서 사용한 단어 

- 항상 결과는 1차원
- 슬라이싱과 다르게 데이터를 새로운 배열로 복사함. (값을 대입하면 색인된 값이 변경되는 구조)



**배열 전치와 축 바꾸기**

배열 전치(transpose) : 데이터를 복사하지 않고 데이터의 모양이 바뀐 뷰를 반환하는 기능

- `transpose` 메서드와 T라는 이름의 특수한 속성을 가짐
- `@` : 행렬 곱셈을 수행하는 방법

- `swapaxes` : 두 개의 축 번호를 받아서 배열을 뒤 바꾼다.

## 3. 유니버설 함수: 배열의 각 원소를 빠르게 처리하는 함수

### 개념정리

**유니버셜 함수**

- `ufnc` : ndarray 안의 데이터 원소별로 연산을 수행하는 함수 
  - 하나 이상의 스칼라 값을 받아서 하나 이상의 스칼라 결괏값을 반환하는 간단한 함수를 빠르게 수행하는 벡터화된 레퍼 함수
- `sqrt`, `exp` : **단항 유니버셜 함수**
- `add, maximum` : **이항 유니버셜 함수**

- `modf` : 분수를 받아서 몫과 나머지를 함께 반환
- 유니버셜 함수는 선택적으로 out 인수를 활용해 계산 결과를 새로운 배열로 만들지 않고 기존 배열에 할당이 가능

## 4. 배열을 이용한 배열 기반 프로그래밍

### 개념정리

- `numpy.meshgrid` : 두 개의 일차원 배열을 받아 모든 (x,y) 짝을 만들 수 있는 2차원 배열 두 개를 반환
- `imshow` 함수를 사용해 2차원 배열 시각화 가능



**배열 연산으로 조건부 표현하기**

- `numpy.where` : **if 조건 else y** 같은 삼항식 벡터화 버전
  - 순수 파이썬으로 수행되기 때문에 큰 ㅐㅂ열은 처리 못하고
  - 다차원 배열에서도 처리를 못한다.



**수학 메서드와 통계 메서드**

- 배열 전체 배열의 한 축에 속하는 자료 : 통계를 계산하는 수학 함수는 배열 클래스의 메서드로 사용이 가능
- 전체의 합, 평균, 표준편차는 넘파이의 최상위 함수 이용 또는 배열의 인스턴스 메서드를 사용
- `cumsum, cumprod` : 중간 계산값을 담고 있는 배열을 반환
  - axis =1 : 모든 열의 평균을 구해라
  - axis = 0 : 행의 합을 구하라 



**불리언 배열을 위한 메서드** : 1(True) or 0(False)로 강제

**정렬** : `sort` 메서드 사용 (내장 리스트 자료형과 마찬가지)

**집합 관련 함수**

- `numpy.unique()` : 함수 배열 내 중복된 원소 제거하고 남은 원소를 정렬된 형태로 반환하는 함수
  - 대부분 넘파이 버전이 더 빠르다. 

**배열 데이터의 파일 입출력** : 텍스트나 바이너리 형식의 데이터 불러오거나 저장이 가능

- `save, load` : 배열 데이터를 디스크에 저장하고 불러오는 함수 
  - `.npy` 파일 형태로 저장
