# 1️⃣ 학습 내용 정리

## 1. 판다스 자료구조 소개

### 개념정리

**판다스 (pandas)** : 고수준의 자료구조와 파이썬을 통한 빠르고 쉬운 데이터 분석 도구를 제공

- 초반에는 넘파이의 배열 기반의 계산 스타일을 많이 차용
- 차이점 : 표 형식의 데이터나 다양한 형태의 데이터를 다루는 데 초점을 맞춰 설계했다는 점



**Series** 

- 일련의 객체를 담을 수 있는 **1차원 배열 같은 구조**
- 색인이라고 하는 배열의 데이터와 연관된 이름을 가짐
- 넘파이 배열과 비교하면, 단일 값을 선택하거나 여러 개의 값을 선택할 때 색인으로 라벨 사용이 가능
  - 불리언 배열 사용하여 값을 걸러내거나, 스칼라 곱셈, 수학 함수를 적용하는 넘파이 배열 연산을 수행해도 색인과 값 연결을 유지 된다. 
- 파이썬 딕셔너리 객체로부터 Series 객체 생성이 가능
- **산술 연산에서 색인과 레이블로 자동 정렬하는 기능이 존재**



**DataFrame**

- 표 같은 스프레드시트 형식의 자료구조
  - 행과 열에 대한 색인을 가지며, 색인의 모양이 같은 Series 객체를 담은 파이썬 딕셔너리
- `head, tail` 을 통해 출력이 가능
- `del` : 열을 삭제가 가능
- 중첩된 딕셔너리를 이용해 DataFrame을 생성할 때, 딕셔너리 값은 키의 값별로 조합되어 결과의 색인이 되지만, 색인을 직접 저장하면 지정된 색인으로 DataFrame을 생성



**색인 객체**

- 축 레이블과 다른 메타데이터를 저장하는 객체
- 색인 중요 메서드
  - `append, difference, intersection, union, isin, delete, drop, insert, is_monotonic, is_unique, unique`

## 2. 핵심 기능

### 개념정리

**재색인** : 새로운 색인에 적합하도록 객체를 새로 생성하는 기능

- `reindex` 호출 : 새로운 색인에 맞게 재배열, 존재하지 않는 색인값이 있으면 Nan으로 새로 추가
- 시계열 같은 순차적 데이터 재색인할때는 `method` 옵션을 이용해서 `ffill`같은 메서드를 통해 누락된 값을 이전의 값으로 채워넣기가 가능
- 특정 축을 재색인하는 방법 : 새로운 축 레이블을 인수로 넘긴 후 다음 `axis` 키워드를 사용해 재색인할 축을 지정하는 것
- `loc` : 재색인이 가능하며 사용자가 이 방법을 선호 



**하나의 행이나 열을 삭제하기**

- `drop` : 선택한 값들이 삭제된 새로운 객체를 얻기가 가능



**색인하기, 선택하기, 거르기**

- 넘파이 배열의 색인과 유사하게 작동, 정수가 아니어도 된다. 
- []정수의 처리 방식이 다르다. loc을 선호하는 경우도 많음. 

**loc, iloc** : 행렬의 부분집합을 선택하기가 가능하다.

- 서로 다른 색인을 가진 객체 간의 산술 연산도 처리가 가능
  - 짝이 맞지 않은 색인은 통일이 된다. 
  - 서로 겹치는 색인이 없는 경우 = 결측처 처리 (**따로 산술연산 처리가 필요**)

**정렬**

- `sort_values` 메서드 사용하기, 결측치는 가장 마지막에 위치
- `np_position` 옵션을 통해 누락된 값을 먼저 정렬하기 가능

## 3. 기술 통계 계산과 요약

### 개념정리

수학 메서드 or 통계 메서드 : 하나의 행과 열에서 단일 값을 구하는 축소 혹은 요약 통계 범주에 해당

- 전체 행과 열이 모든 **NA**값이면 그 합은 0
  - 값이 하나라도 NA이면 결괏값은 NA 
- `idxmin, dixmax` : 최소 혹은 최댓값을 가진 색인값 같은 간접 통계값을 가져옴.
- `cumsum` : 누산된 값 제공
- `describe` : 한 번에 여러개의 요약 통계를 가져옴. 



**상관관계, 공분산**

- 두 쌍의 인수가 필요. 
  - `MSFT` , `corr, cov` 메서드를 통해 상관관계와 공분산을 계산

**유일값, 값 세기, 멤버십**

- 유일값 : `uniques.sort()`  정렬된 순서로 가져오기
- `value_counts()` : Seires에서 도수를 계산해서 반환
